== Description ==

Trinay Zencoder plugin can be used to transcode videos in amazon S3 using Zencoder.

You need to provide amazon s3 access key and zencoder access key in the settings.

Trinay Zencoder plugin provides a page for starting the process of transcoding videos from amazon s3. You can see the filenames of transcoded videos in Wordpress. 
The Plugin avoids duplicate transcoding. Trinay Zencoder has implemented notification, so you can see the video conversion status on plugin home page. 

Option to play videos.

Example Short Code:
[TTS_Video id=25]
id - video id showing on home screen of the Trinay Zencoder Plugin.

== Installation ==

1. Unzip Trinay-Zencoder.zip
2. Upload `Trinay-Zencoder` folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
